### Membuat Project Baru

1. Buat direktori baru bernama `undangan-pesta`, kemudian pada folder tersebut jalankan perintah berikut untuk menginisialisasi project barun dengan golang.

```sh
$ go mod init undanganpesta
```

2. Buat file bernama `main.go`, tuliskan kode berikut:

```go showLineNumbers
package main

import "fmt"

func main()  {
  fmt.Println("membuat fitur")
}
```

Kita dapat dapat menggunakan _depedency package_ dengan menggunakan `import` _statement_. Fungsi `main()` merupakan _entry point_ pada aplikasi. fmt _package_ merupakan _standard library_ yang disediakan Go.

3. Menjalankan program

```bash
$ go run .
```
